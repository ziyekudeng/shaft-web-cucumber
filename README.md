# shaft-web-cucumber
SHAFT_ENGINE 测试用例

## 项目配置信息
- JDK版本: 21
- Maven版本: 3.9.12
- SHAFT_ENGINE版本: 9.4.20251220

## 项目依赖配置 (pom.xml)

## 依赖说明
- SHAFT_ENGINE: 核心测试框架，版本9.4.20251220
- aspectjweaver: AOP支持库，版本1.9.24
- surefire-testng: 测试执行引擎，版本3.5.4

## 配置说明
- 编译器插件: maven-compiler-plugin 3.11.0，支持JDK 21
- 资源插件: maven-resources-plugin 3.3.1
- 测试插件: maven-surefire-plugin 3.5.4，集成TestNG支持
